# Objectifs:
* Fournir aux personnes concernés les emplacements des stationemments handicapés
* Proposer un itinéraire vers la place handicapé la plus proche

# Technologies

* Ionic
* Vue JS
* MongoDB (pour le stockage)
* Leaflet (affichage de la carte)
